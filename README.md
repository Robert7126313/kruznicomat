# Možná a zajímavá rozšíření
1. Výběr barvy kružnice **HOTOVO** + náhled barvy
2. SpinBox na velikost kružnice **HOTOVO**
3. Funkce zpět
4. Když klepnu pravým tlačítkem myši, mám kružnici vyplněnou **HOTOVO pro kružnici** 
5. Při drag-and-drop nastavit velikost kružnice
6. Přidat další nástroje
   1. Elipsa (nejen kružnice). **HOTOVO**
   2. Polygon – možnost klikáním vytvářet vlastní tvary. **ZATÍM JEN ČTVEREC**
   3. Křivky – Beziérovy spliny. **HOTOVO**
   4. Nástroj pro text – vložení textu do obrázku.
   5. Nástroj gumy (pro pixelový i vektorový styl, podle typu tvé aplikace).
7. Ukládání obrázku

# Radim doplní

9. Možnost zapnout a vypnout vodící mřížku (třeba po 10px) **HOTOVO**
10. Umožnit v nastavení nastavit krok vodící mřížky,

# Všeobecné rozšíření
1. Zachytávání na mřížku (snap-to-grid) – objekty se zarovnají na nejbližší grid. 
2. Zobrazování náhledového „ducha“ objektu při kreslení (preview tvaru, než se potvrdí). **HOTOVO**
3. Výběr objektu a jeho následná editace – změna velikosti, barvy, posunutí.
4. Skupinové označení objektů – výběrový rámeček (drag to select).
5. Mazání objektu klávesou Delete.
6. Změna tloušťky čáry (slider nebo selection box).
7. Klávesové zkratky – třeba Ctrl+Z, Ctrl+S, nástroje apod.

# okročilejší funkce
1. Vrstvy (Layers) – podobně jako v grafických editorech. 
2. Export do různých formátů – PNG, JPG, SVG (vektory).
3. Import obrázků jako pozadí – možnost přes ně kreslit.
4. Režim pravítka – měření vzdáleností mezi body.
5. Zobrazit ohraničující box objektu při jeho výběru.
6. Magnetické vodicí linky – automaticky se zobrazí u zarovnání s ostatními objekty.
7. Historie akcí (undo/redo) v podobě seznamu – možnost vrátit se ke konkrétnímu kroku.

# Uživatelské prostředí
1. Tmavý / světlý režim.
2. Možnost přesouvat panely a docky (toolbar, barva, vrsty).
3. Ukládání a načítání konfigurace aplikace (např. poslední použité barvy, nástroje).

# Speciální funkce
1. Duplikace objektu (Ctrl+D).
2. Zarovnání objektů – k sobě navzájem nebo ke stránce (nahoru, doprostřed, vlevo, …).
3. Rotace objektů – ručně nebo numericky. 
4. Zamykání objektů – aby nešly posouvat ani měnit.
5. Kontextové menu po pravém kliknutí – specifické akce pro označený objekt.
